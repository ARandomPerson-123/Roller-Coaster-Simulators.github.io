Top Thrill 2

This roller coaster is the tallest operating roller coaster ( as of December 15th 2025 )
This Roller coaster is 420 ft tall, goes 120 MPH, has 0 inversions, has a 2 minute ride time.
This Roller coaster used to be named "Top Thrill Dragster" and used to have an Hydraulic Launch system, but was removes after an incident,
so it now has a triple LSM launch system where it swings back and forth to be able to get over the tophat.
